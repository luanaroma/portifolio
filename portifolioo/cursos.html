Curso de Informática

O curso de Informática ensina os fundamentos do uso de computadores, redes, programação e ferramentas digitais. Ele desenvolve habilidades práticas e teóricas essenciais para o uso consciente da tecnologia e para quem deseja seguir carreira na área da computação.

Instituição: Escola Estadual Presidente Dutra

Período: [fevereiro/2024] - Dezembro/2026]

Descrição: Curso básico de programação abordando sintaxe, variáveis, operadores, controle de fluxo e estruturas de dados.

Desenvolvimento Web com HTML, CSS e JavaScript

Descrição: Curso sobre os fundamentos do desenvolvimento web, incluindo criação de páginas responsivas e interativas utilizando HTML, CSS e JavaScript.
